# SPOTnSORT – Smart Civic Issue Reporting & Tracking Platform

SPOTnSORT is a citizen-powered web platform that enables users to report civic issues such as potholes, garbage overflow, broken streetlights, and water leaks. The platform bridges the gap between citizens and authorities by providing a transparent system for reporting, tracking, and resolving urban problems.

It promotes community engagement and accountability through real-time reporting, dashboards, and a reward-based participation system.

---

## Features

### Citizen Features
- User registration and login
- Report civic issues with description and location
- Upload geo-tagged images of problems
- Track report status (Pending, In Progress, Completed)
- View reported issues on an interactive map
- Earn reward points for reporting issues
- View personal dashboard with report statistics

### Authority Features
- Authority registration and verification
- Dashboard showing assigned reports
- Update issue status and schedule tasks
- Upload proof images after issue resolution
- View analytics and performance metrics
- Manage and monitor civic complaints efficiently

## System Workflow

1. User registers and logs in.
2. Citizen submits a civic issue with location and description.
3. System assigns the report to the relevant authority.
4. Authority reviews and updates the issue status.
5. Issue progress is tracked through dashboards.
6. Users receive updates and reward points for participation.

The workflow includes user reporting, automatic assignment, dashboard tracking, analytics, and a rewards system for engagement.
